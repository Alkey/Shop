# jv-internet-shop

****Application functionality:

* Registration;
* Login, logout;
* User, Admin roles;
* User's shopping cart;
* Product addition to the store;
* Removing products from store;
* User order  creation;
* View orders;
* User filtering by RBAC;

****This project used technologies:

* Java 8;
* Tomcat;
* JDBC;
* Maven;
* Servlets;
* JSP;
* MySQL;
* JSTL;
